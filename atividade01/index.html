Bem-vindo

Este é um site base para atividade de responsividade.

HTML

HTML (HyperText Markup Language) é a linguagem responsável por estruturar o conteúdo das páginas web, definindo elementos como títulos, parágrafos, links e imagens.

CSS

CSS (Cascading Style Sheets) é a linguagem utilizada para estilizar páginas web, controlando cores, layout, fontes e tornando o design responsivo.

JavaScript

JavaScript é a linguagem de programação que adiciona interatividade às páginas web, permitindo criar comportamentos dinâmicos como animações e respostas a eventos.
